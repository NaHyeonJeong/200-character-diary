# Introduce "200 character diary"

## 기능
### 일기 입력 페이지
1. 날짜를 선택해 특정 날짜에 일기를 작성
2. 기분 정도를 체크할 수 있음
3. 글자수는 200자로 제한해서 입력 가능
### 일기 리스트 페이지
1. 작성했던 일기를 ListView로 보여줌
2. 일기를 저장, 메인 페이지로 이동 버튼 있음
### 지난 일기 확인 페이지
1. 예전에 작성했던 일기를 볼 수 있음
2. 메인 화면과 일기 리스트 페이지로 이동 가능한 버튼 있음
